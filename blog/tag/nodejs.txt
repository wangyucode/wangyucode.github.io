иҖҢNodeJSдёҖзӣҙдҫқиө–йҮҮз”ЁCommonJSзҡ„жЁЎеқ—еҢ–ж–№жЎҲпјҢеңЁжңҖиҝ‘еҸ‘еёғзҡ„зүҲжң¬дёӯд№ҹејҖе§Ӣж”ҜжҢҒems, з”ұдәҺе·ЁеӨ§зҡ„еҺҶеҸІеҢ…иўұпјҢNodeJSе№¶жІЎжңүжҠӣејғCommonJSпјҢжүҖд»ҘеңЁNodeJSдёӯе®һйҷ…ж”ҜжҢҒдёӨз§ҚжЁЎеқ—еҢ–ж–№ејҸпјҢesmе’ҢCommonJS.

жӯӨеӨ–пјҢзҺ°еңЁеҫҲеӨҡеүҚз«Ҝе·ҘзЁӢдҪҝз”ЁTypeScriptејҖеҸ‘пјҢиҖҢTSзҡ„жЁЎеқ—и§Јжһҗж–№ејҸд№ҹйңҖиҰҒиҝӣиЎҢдёҖдәӣй…ҚзҪ®пјҢеңЁиҝҷ3иҖ…з»“еҗҲзҡ„иҝҮзЁӢдёӯжңүеҫҲеӨҡеқ‘пјҢжң¬ж–Үд»Ӣз»Қе°ҶдёүиҖ…е®ҢзҫҺз»“еҗҲзҡ„жңҖдҪіе®һи·өгҖӮ

## еҗҢж—¶еј•е…ҘCommonJSеҢ…е’ҢESMеҢ…

дёәдәҶжөӢиҜ•жҲ‘д»¬еҗҢж—¶еј•е…Ҙ`lodash`зҡ„CommonJSзүҲжң¬е’ҢESMзүҲжң¬

```bash
npm i lodash-es lodash
```

еңЁNodeJSдёӯпјҢжЁЎеқ—дҪңиҖ…йҷӨйқһжҳҫзӨәжҢҮе®ҡдҪҝз”Ёesmж–№ејҸеҠ иҪҪжЁЎеқ—пјҡ

- дҪҝз”Ё`.mjs`
- еңЁ`package.json`дёӯзҡ„`type`еӯ—ж®өжҢҮе®ҡдёә`module`
- дҪҝз”Ё`--input-type`ж Үи®°

еҗҰеҲҷй»ҳи®ӨеҠ иҪҪж–№ејҸдёәCommonJSгҖӮ

иҖҢдёҠйқўзҡ„дҫӢеӯҗдёӯ`lodash-es`зҡ„`package.json`дёӯзҡ„`type`е°ұжҳҜ`module`гҖӮ

## дҪҝз”ЁCommonJSеҢ…

еҲӣе»әдёҖдёӘ`common-js.js`ж–Үд»¶пјҡ

```javascript
const _ = require('lodash');
console.log(_.VERSION); // print '4.17.21'
```

д»ҘдёҠжҳҜCommonJSзҡ„еҶҷжі•пјҢеҫҲд№ д»ҘдёәеёёпјҢиғҪеӨҹжӯЈзЎ®жү“еҚ°гҖӮ

## дҪҝз”ЁesmеҢ…

еҲӣе»әдёҖдёӘ`esm.js`ж–Үд»¶пјҡ

```javascript
import _ from 'lodash-es';
console.log(_.VERSION);
```

з¬¬1дёӘеқ‘еҮәзҺ°дәҶ, жҠҘиҜӯжі•й”ҷиҜҜпјҡ`SyntaxError: Cannot use import statement outside a module`

иҜҙж— жі•еңЁжЁЎеқ—еӨ–йғЁдҪҝз”ЁimportиҜӯеҸҘпјҢдёҖж—ҰдёҖдёӘж–Үд»¶жңүйЎ¶зә§(top-level)зҡ„`import`жҲ–`export`пјҢе®ғдјҡиў«еҪ“дҪңдёҖдёӘжЁЎеқ—пјҢеҗҰеҲҷжҳҜдёҖдёӘи„ҡжң¬ж–Үд»¶гҖӮ

жүҖд»ҘжҲ‘д»¬йңҖиҰҒжҠҠиҝҷдёӘж–Үд»¶еҸҳжҲҗesmпјҢдёӨз§Қж–№ејҸпјҡ

1. йҮҚе‘ҪеҗҚж–Үд»¶пјҢдҝ®ж”№жү©еұ•еҗҚдёә`.mjs`;
2. еңЁ`package.json`дёӯж·»еҠ `"type": "module"`

дҪҶеҰӮжһңйҮҮз”ЁйҖүйЎ№2пјҢз¬¬2дёӘеқ‘е°ұдјҡеҮәзҺ°дәҶ, еӣһеҺ»иҝҗиЎҢ`common-js.js`жҠҘй”ҷпјҡ`ReferenceError: require is not defined in ES module scope, you can use import instead`гҖӮ
иҜҙ`require`еңЁesmдёӯдёҚж”ҜжҢҒпјҢжӯӨж—¶еҸҜд»ҘйҮҚе‘ҪеҗҚ`common-js.js`еҗҺзјҖдёә`.cjs`пјҢе‘ҠиҜүжЁЎеқ—еҠ иҪҪеҷЁпјҢжӯӨж–Үд»¶д»ҘCommonJSеҠ иҪҪгҖӮ

## еј•е…ҘTypescript

е®үиЈ…TSпјҡ

> ts-node еҸҜд»ҘеңЁnodejsдёӯзӣҙжҺҘиҝҗиЎҢtsж–Үд»¶иҖҢж— йңҖзј–иҜ‘

з„¶еҗҺе°ҶеҲҡжүҚзҡ„`esm.js`е’Ң`common-js.cjs`жү©еұ•еҗҚж”№дёә`.ts`

дҪҝз”Ё`ts-node`е‘Ҫд»ӨзӣҙжҺҘиҝҗиЎҢ`esm.ts`

```bash
npx ts-node esm.ts
```

з¬¬3дёӘеқ‘еҮәзҺ°дәҶпјҢжү©еұ•еҗҚдёҚж”ҜжҢҒпјҡ`TypeError [ERR_UNKNOWN_FILE_EXTENSION]: Unknown file extension ".ts" for xxx`

иҝҷжҳҜеӣ дёәesmзҡ„й»ҳи®Өжү©еұ•еҗҚжҳҜjsиҖҢдёҚжҳҜtsпјҢжӯӨж—¶жҲ‘д»¬йңҖиҰҒпјҡ

1. `ts-node --esm`
2. ж·»еҠ  `"ts-node": {"esm": true}`еҲ°`tsconfig.json`гҖӮ

еҰӮжһңйҮҮз”Ёж–№жЎҲ1пјҢз¬¬4дёӘеқ‘еҮәзҺ°дәҶпјҢж— жі•жүҫеҲ°жЁЎеқ—пјҡ`error TS7016: Could not find a declaration file for module 'lodash-es'`гҖӮ

жӯӨж—¶ж·»еҠ й…ҚзҪ®: `"esModuleInterop": true`еҲ°`tsconfig.json`зҡ„`compilerOptions`дёӯгҖӮ

ж·»еҠ е®ҢжҲҗеҗҺз¬¬5дёӘеқ‘еҮәзҺ°дәҶпјҢеј•з”Ёй”ҷиҜҜпјҡ`ReferenceError: exports is not defined in ES module scope`

жӯӨж—¶ж·»еҠ й…ҚзҪ®`module": "ESNext"`еҲ°`tsconfig.json`зҡ„`compilerOptions`дёӯгҖӮ

ж·»еҠ е®ҢжҲҗеҗҺз¬¬6дёӘеқ‘еҮәзҺ°дәҶпјҢжүҫдёҚеҲ°жЁЎеқ—пјҡ`ReferenceError: exports is not defined in ES module scope`

жӯӨж—¶й…ҚзҪ®жЁЎеқ—зҡ„и§Јжһҗж–№ејҸдёәжңҖж–°зҡ„`"moduleResolution": "Node16"`гҖӮ

еҲ°жӯӨз»ҲдәҺеҸҜд»ҘжҲҗеҠҹиҝҗиЎҢ`esm.ts`дәҶгҖӮ

иҝ”еӣһеҺ»иҝҗиЎҢ`common-js.ts`:

```bash
npx ts-node common-js.ts
```

з¬¬7дёӘеқ‘еҮәзҺ°дәҶпјҢе®ғжҠҠжҲ‘д»¬зҡ„CommonJSжЁЎеқ—еҪ“esmдәҶпјҡ`ReferenceError: require is not defined in ES module scope, you can use import instead`

## й•ңеғҸеҲҶеҸ‘

еҜ№дәҺDockerжһ„е»әзҡ„й•ңеғҸпјҢжҲ‘д»¬еҸҜд»Ҙе°Ҷе…¶жҺЁйҖҒеҲ°Docker hubд№Ӣзұ»зҡ„й•ңеғҸд»“еә“гҖӮ
з„¶еҗҺеңЁжңҚеҠЎеҷЁдёҠиҝҗиЎҢ`docker pull <Image Name>`жҲ–иҖ…`docker-compose pull <Service>`е°ұеҸҜд»Ҙе°Ҷй•ңеғҸжӢүдёӢжқҘгҖӮ
з„¶еҗҺеҶҚйҮҚж–°иҝҗиЎҢе°ұеҸҜд»ҘдәҶгҖӮ

иҝҷз§Қж–№ејҸи§ЈеҶідәҶжңҚеҠЎеҷЁзҡ„е®үе…Ёй—®йўҳгҖӮ

дҪҶжҳҜеҰӮдҪ•е°ҶиҝҷдёҖеҘ—жөҒзЁӢиҮӘеҠЁеҢ–е‘ўгҖӮ

## иҮӘеҠЁжЈҖжҹҘй•ңеғҸжӣҙж–°е’ҢйҮҚж–°иҝҗиЎҢ

дёҖдёӘжҖқи·Ҝе°ұжҳҜдё“й—ЁеҶҷдёҖдёӘе®ҲжҠӨзЁӢеәҸпјҢеңЁжҺҘж”¶еҲ°жҹҗз§ҚAPIиҜ·жұӮж—¶пјҢжү§иЎҢд»ҘдёҠзҡ„ж“ҚдҪңгҖӮдҪҶжҳҜиҝҷж ·е®ғе°ұи„ұзҰ»дәҶDockerзҡ„з®ЎзҗҶпјҢиҖҢдё”е…¶жң¬иә«д№ҹжҳҜйңҖиҰҒз»ҙжҠӨе’ҢйғЁзҪІзҡ„гҖӮ

дҪҶжҳҜеҰӮжһңе°Ҷе®ғд№ҹдҪңдёәDockerй•ңеғҸзҡ„иҜқпјҢ`Container`еҸҲеҰӮдҪ•жҺ§еҲ¶е®ҝдё»жңәдёӯзҡ„`Docker Engine`е‘ўпјҹ

иҝҷд№ҹжңүеҠһжі•пјҢжҲ‘д»¬еҸӘйңҖиҰҒе°Ҷ`Host`зҡ„DockerеҹҹеҘ—жҺҘеӯ—(`UNIX domain socket`)пјҢжҢӮиҪҪеҲ°`Container`дёӯпјҡ

```bash
docker run ... -v /var/run/docker.sock:/var/run/docker.sock
```

еӣ дёәDockerжҳҜCSжЁЎејҸпјҢиҝҷж ·`Container`е°ұеҸҜд»ҘжҺ§еҲ¶е®ҝдё»жңәзҡ„Dockerеј•ж“ҺпјҢд»Ҙе®һзҺ°дёҠиҝ°ж“ҚдҪңдәҶгҖӮжңүдәәе°Ҷе…¶з§°дёә`Docker in Docker`гҖӮ

иҖҢд»ҘдёҠиҝҷдёӘжҖқи·Ҝе·Із»ҸжңүдәәйҖ еҘҪиҪ®еӯҗдәҶгҖӮ

## Github Marketplace

з»„еҗҲеҸҜд»ҘжҳҜеӨҡз§ҚеӨҡж ·зҡ„пјҢдҪҶжҳҜиҝҷдәӣ`Action`йҖҡеёёйғҪжҳҜзұ»дјјзҡ„гҖӮжүҖд»Ҙ Github еҲ©з”ЁиҮӘе·ұејҖжәҗзҡ„зү№зӮ№пјҢжүҖжңүдәәйғҪеҸҜд»Ҙе°ҶиҮӘе·ұзҡ„`Action`еҸ‘еёғеҲ°`Github Marketplace`дёӯпјҢдҫӣд»–дәәдҪҝз”ЁгҖӮ

иҝҷж„Ҹе‘ізқҖпјҢдҪ иғҪжғіеҲ°зҡ„`Action`еҮ д№ҺйғҪе·Із»Ҹиў«еҶҷеҘҪдәҶпјҢзӣҙжҺҘжӢҝжқҘз”Ёе°ұиЎҢдәҶпјҢдёҚз”ЁйҮҚеӨҚйҖ иҪ®еӯҗгҖӮ

еҪ“з„¶иҮӘе·ұеҶҚйҖ дёҖдёӘиҪ®еӯҗд№ҹжҳҜеҫҲз®ҖеҚ•зҡ„гҖӮ

## дёҖдёӘе…ёеһӢзҡ„ workflow

```yml
name: Node.js CI

on: [push] # pushд»Јз Ғж—¶жү§иЎҢ

jobs:
  build: # jobsеҗҚз§°
    runs-on: ubuntu-latest # ж“ҚдҪңзі»з»ҹ

    strategy:
      matrix:
        node-version: [8.x, 10.x, 12.x] # nodeзүҲжң¬

    steps:
      - uses: actions/checkout@v2 # еј•з”ЁactionпјҡжӢүд»Јз Ғ
      - name: Use Node.js ${{ matrix.node-version }} # еј•з”Ёactionпјҡй…ҚзҪ®nodeзҺҜеўғ
        uses: actions/setup-node@v1
        with:
          node-version: ${{ matrix.node-version }}
      - run: npm install # и·‘е‘Ҫд»Өпјҡе®үиЈ…дҫқиө–
      - run: npm run build --if-present # и·‘е‘Ҫд»Өпјҡе®үиЈ…дҫқиө–
      - run: npm test # и·‘е‘Ҫд»Өпјҡжү§иЎҢжөӢиҜ•
```

## дҪҝз”ЁеҜҶз ҒеҸӮж•°

## е…ідәҺе“ҲеёҢи·Ҝз”ұ

зҺ°д»Јзҡ„еүҚз«ҜжЎҶжһ¶йғҪжҳҜеҚ•йЎөйқўзҡ„(single-page application)пјҢеҚіз”ЁжҲ·и®ҝй—®зҡ„ж°ёиҝңжҳҜж №зӣ®еҪ•зҡ„index.htmlпјҢиҝҷж—¶зҪ‘йЎөең°еқҖеҰӮдҪ•еұ•зҺ°е°ұжҳҜдёҖдёӘй—®йўҳгҖӮи§ЈеҶіж–№жЎҲжңү2дёӘпјҡ

- еүҚз«Ҝи§ЈеҶіпјҡдҪҝз”Ёе“ҲеёҢи·Ҝз”ұпјҢеҚіз”ЁжҲ·зңӢеҲ°зҡ„ең°еқҖж ҸURLжҳҜ`#`еҲҶеүІзҡ„пјҢеҰӮ`https://wycode.cn/admin/#/users/123`пјҢе®һйҷ…и®ҝй—®д»Қз„¶жҳҜ`index.html`гҖӮ`UmiJS`й…ҚзҪ®еҰӮдёӢ

```js
export default {
    В·В·В·
    history: 'hash'
    В·В·В·
} as IConfig;  
```

- еҗҺз«Ҝи§ЈеҶіпјҡдҪҝз”ЁйҮҚе®ҡеҗ‘пјҢеҚіи®ҝй—®indexд»ҘеӨ–зҡ„е…¶е®ғең°еқҖпјҢйғҪйҮҚе®ҡеҗ‘еҲ°`index.html`гҖӮ`nginx`й…ҚзҪ®еҰӮдёӢ:

```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```

## е…ідәҺеҢәеқ—`block`

еҢәеқ—еҸҜд»ҘзҗҶи§ЈдёәйЎөйқўжЁЎжқҝпјҢиҝҷдёӘжЁЎжқҝжҳҜеңЁ`ant-design-pro`зҡ„йў„и§ҲзҪ‘з«ҷдёӯпјҢеҰӮжһңдҪ зңӢдёӯдәҶжҹҗдёӘеҠҹиғҪпјҢпјҲй»ҳи®ӨеҲӣе»әзҡ„е·ҘзЁӢжҳҜз©әзҷҪзҡ„пјҢеҸӘжңүдёҖдёӘWelcomeйЎөйқўпјүзӮ№еҮ»еҸідёӢи§’зҡ„дёӢиҪҪжҢүй’®пјҢдјҡеј№еҮәдёҖдёӘе‘Ҫд»ӨпјҢжү§иЎҢиҝҷдёӘе‘Ҫд»Өе°ұиғҪе°ҶиҝҷдёӘйЎөйқўеҠ е…ҘеҲ°дҪ иҮӘе·ұзҡ„е·ҘзЁӢдёӯпјҢйқһеёёж–№дҫҝгҖӮе…¶е®һиҙЁжҳҜжү§иЎҢдәҶ`git clone`е…ӢйҡҶйў„и§Ҳе·ҘзЁӢзҡ„йғЁеҲҶж–Үд»¶гҖӮ

жҜ”еҰӮпјҡж·»еҠ дёҖдёӘзұ»дјјйў„и§Ҳе·ҘзЁӢзҡ„404йЎөйқў

```bash
npx umi block add Exception404  --path=/exception/404
```

з»ҸиҝҮе°қиҜ•пјҢжҲ‘еҸ‘зҺ°зӣ®еүҚеҢәеқ—еј•е…Ҙж—¶пјҢе№¶дёҚжҳҜеҗҲе№¶ж–Үд»¶пјҢиҖҢжҳҜеҲӣе»әдёҖдёӘзӣ®еҪ•пјҢйҮҢйқўеҢ…жӢ¬дәҶиҝҷдёӘйЎөйқўз”ЁеҲ°зҡ„жүҖжңүз»„д»¶пјҢз”ҡиҮіжҳҜеӣҪйҷ…еҢ–ж–Үжң¬йғҪжҳҜеҚ•зӢ¬зҡ„дёҖеҘ—гҖӮдёҚз®ЎдҪ е·ҘзЁӢеҶ…жңүжІЎжңүиҝҷдәӣз»„д»¶пјҢе®ғйғҪдјҡйҮҚж–°еҲӣе»әдёҖеҘ—гҖӮ

жҜ”еҰӮеҲҡеҠ е…ҘдәҶ404йЎөйқўеҸҲжғіеҠ е…Ҙ403йЎөйқўпјҢжң¬е·Із»ҸеӯҳеңЁзҡ„Exceptionз»„д»¶пјҢдјҡеңЁ403ж–Үд»¶еӨ№еҶҚеҲӣе»әдёҖд»ҪдёҖжЁЎдёҖж ·зҡ„еүҜжң¬пјҢиҝҷзӮ№пјҢжҲ‘и§үеҫ—жҳҜдёҚеҸҜжҺҘеҸ—зҡ„гҖӮ

иҝҷж ·зҡ„иҜқжҲ‘жӣҙеҖҫеҗ‘дёҺеҸӮиҖғblockзҡ„е®һзҺ°жүӢеҠЁеҠ е…ҘйңҖиҰҒзҡ„ж–Үд»¶пјҢиҖҢдёҚжҳҜдҪҝз”Ё`umi block add`еҠҹиғҪпјҢе®ҳж–№зҡ„blockеңЁиҝҷйҮҢпјҡhttps://github.com/ant-design/pro-blocks

> йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҡTravis CIзӣ®еүҚжңүдёӨдёӘеҹҹеҗҚ`travis-ci.org`е’Ң`travis-ci.com`пјҢжҺЁиҚҗдҪҝз”Ё`com`пјҢе®ҳж–№е·Із»Ҹе°Ҷ`.org`еҗҲе№¶еҲ°`.com`дәҶгҖӮ

## й…ҚзҪ®

Travis CIйҖҡиҝҮиҜ»еҸ–йЎ№зӣ®ж №зӣ®еҪ•зҡ„`.travis.yml`ж–Үд»¶иҺ·еҸ–й…ҚзҪ®дҝЎжҒҜпјҢжүҖд»ҘйңҖиҰҒеңЁе·ҘзЁӢзӣ®еҪ•дёӢй…ҚзҪ®`.travis.yml`

```yml
# safelist
branches:
  only:
  - master
language: node_js
node_js:
  - "8"
script: ./node_modules/.bin/hexo clean && ./node_modules/.bin/hexo generate
deploy:
  # deploy to github pages
  - provider: pages
    skip_cleanup: true
    target_branch: master
    github_token: $GITHUB_TOKEN  # Set in the settings page of your repository, as a secure variable
    repo: wangyucode/wangyucode.github.io
    local_dir: public
after_deploy:
  - node deploy/sftp.js $SERVER_PASSWORD
```

- `branches`еҸҜд»Ҙй…ҚзҪ®йңҖиҰҒжһ„е»әзҡ„й»‘зҷҪеҗҚеҚ•пјҢжҲ‘иҝҷйҮҢи®ҫзҪ®дәҶзҷҪеҗҚеҚ•пјҢеҸӘжһ„е»ә`master`еҲҶж”Ҝ
- иҜӯиЁҖзҺҜеўғйҖүжӢ©`nodejs`
- дёҖдёӘJobеҢ…еҗ«дёӨдёӘдё»иҰҒйғЁеҲҶпјҡ`install`е’Ң`script`
- жҲ‘иҝҷйҮҢжҢҮе®ҡдәҶnodeзүҲжң¬дёәv8
- `install`й»ҳи®Өдјҡжү§иЎҢ`npm install`пјҢеңЁжңү`yarn.lock`зҡ„е·ҘзЁӢдёӯдјҡжӣҝд»Ј`npm`дёә`yarn`
- `script`дјҡеңЁ`install`е®ҢжҲҗеҗҺжү§иЎҢ
- `deploy`дёӯжҲ‘йғЁзҪІдәҶgithub pagesпјҢ`skip_cleanup`иЎЁзӨәи·іиҝҮcleanupеӣ дёәйғЁзҪІж—¶жҲ‘дёҚйңҖиҰҒжё…зҗҶд»»дҪ•ж–Үд»¶пјҢ`target_branch`жҢҮе®ҡдәҶиҰҒжҸҗдәӨзҡ„еҲҶж”ҜпјҢ`github_token`еӣ дёәжҜ”иҫғж•Ҹж„ҹпјҢжүҖд»ҘдҪңдёәеҸҳйҮҸй…ҚзҪ®еңЁ`Repository Settings`дёӯпјҢ`repo`й…ҚзҪ®дәҶиҰҒжҸҗдәӨзҡ„д»“еә“пјҢ`local_dir`иЎЁзӨәеҸӘдјҡйғЁзҪІ`public`дёӯзҡ„ж–Үд»¶
- `after_deploy`еңЁйғЁзҪІе®ҢжҲҗд№ӢеҗҺжҲ‘иҝҳдјҡе°Ҷж–Үд»¶йғЁзҪІеҲ°жҲ‘зҡ„linuxжңҚеҠЎеҷЁпјҢиҝҷйҮҢз”ЁдәҶ`sftp-sync-deploy`ејҖжәҗеә“
